PyProtect
Termos e politicas

Observe que esta não é uma ferramenta de criptografia avançada e destina-se apenas a impedir que seu código seja lido por quem não conhece programação muito bem. É bem possível descriptografar os códigos usando métodos básicos, então não use esta ferramenta para ocultar informações importantes nem pense que seu código nunca será quebrado.

- Não somos responsáveis pelos códigos processados com o pyprotect.
- Você deve salvar uma cópia do arquivo original antes de usar nossa ferramenta.
- Não temos permissão para descriptografar nenhum arquivo ofuscado pelo pyprotect.
- Todos os arquivos enviados e processados por nosso sistema são excluídos a cada hora e não salvaremos nenhuma cópia deles.
- É proibido criar mecanismos de engenharia reversa para decodificar arquivos processados por nossa ferramenta.
- Você não deve usar o pyprotect para criptografar código malicioso ou impróprio.

Divirta-se!
